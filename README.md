# Template Vite.js

## Présentation

Ce repo est un kit de démarrage de projet Front-end avec la stack suivante :
- Vite.js
- React
- Typescript
- React-Query
- React-Router-Dom
- TailwindCSS

## Installation

Pour demarrer un projet avec ce template, il vous faudra degit :

```bash
npm instal -g degit
```

Et l'utiliser :
```bash
npx degit ElDucche/vite-rrd-tlwd-rq my-app
cd my-app
pnpm install
pnpm dev
```
